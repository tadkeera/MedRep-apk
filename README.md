# MedRep-apk 📱

تطبيق **Med Rep** — نظام CRM/SFA ذكي لمندوبي الدعاية الطبية، يعمل بدون إنترنت، مع تطبيق أندرويد APK جاهز للتثبيت.

## 📦 محتويات المستودع

| المسار | الوصف |
|---|---|
| **`MedRep.apk`** | ✅ ملف APK جاهز للتثبيت مباشرة (v1.1 — versionCode 2) |
| **`Med-Rep-Waleed/`** | كود الويب المصدر (React + Vite + Tailwind) — يشمل صفحة **التحليلات** الجديدة |
| **`android-app/`** | مشروع الأندرويد (WebView + خادم محلي + جسر الحفظ) مع سكربت بناء `build-apk.sh` |
| **`medrep.keystore`** | مفتاح توقيع APK (كلمة السر: `medrep123`) — ضروري لإصدار تحديثات تُثبّت فوق النسخة الحالية |

## ✨ مزايا التطبيق

- 🗂️ إدارة فواتير عينات الأدوية بنظام FIFO
- 🩺 سجل الأطباء والزيارات الميدانية مع تتبع GPS
- 🗺️ خرائط تفاعلية (Leaflet/OpenStreetMap)
- 📊 **صفحة التحليلات الذكية** (تعمل أوفلاين بالكامل):
  - **الخطة الشهرية الذكية**: مبنية على مواقع الأطباء، الكلاس (A/B/C)، عدد الزيارات السابقة، وتحديد الأطباء الذين يجب تكثيف زياراتهم
  - **تحليل الأطباء**: اكتب اسم الطبيب لتقرير شامل (كل الزيارات + العينات المصروفة + نصائح تحسين)
  - **اسألني**: مساعد يجيب على أي سؤال عن بياناتك ويقدم نصائح لتحسين العمل
- 💾 نسخ احتياطي تلقائي إلى `/Med Rep/backup/` وحفظ الملفات والتقارير إلى `/Med Rep/download/`
- 🌐 واجهة ثنائية اللغة (عربي / إنجليزي)

## 📲 تثبيت الـ APK

1. حمّل `MedRep.apk` من هذا المستودع
2. فعّل "السماح بالتثبيت من مصادر غير معروفة" في إعدادات الهاتف
3. ثبّت التطبيق وامنحه صلاحيات **الموقع** و**التخزين** عند أول تشغيل

## 🛠️ البناء من المصدر

### بناء واجهة الويب:
```bash
cd Med-Rep-Waleed
npm install
npx vite build
rm -rf ../android-app/assets/www
cp -r dist ../android-app/assets/www
```

### بناء الـ APK (بدون Android Studio):
```bash
cd android-app
cp ../medrep.keystore .
export ANDROID_SDK_ROOT=/path/to/android-sdk   # يتطلب build-tools;34.0.0 + platforms;android-34
bash build-apk.sh
# الناتج: MedRep.apk
```

> ⚠️ **مهم**: استخدم دائماً نفس `medrep.keystore` عند البناء حتى تُقبل التحديثات فوق النسخة المثبتة بدون فقدان البيانات.

## 🔐 الصلاحيات

- `ACCESS_FINE_LOCATION` / `ACCESS_COARSE_LOCATION` — تتبع GPS للزيارات
- `READ/WRITE_EXTERNAL_STORAGE` + `MANAGE_EXTERNAL_STORAGE` — مجلدات backup/download
- `INTERNET` — الخرائط والخطوط
